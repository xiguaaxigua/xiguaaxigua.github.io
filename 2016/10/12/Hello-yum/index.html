Hello yum!

10/12/2016 JavaScript

yum即 Shell前端软件包管理器（Yellow dog Updater, Modified），类似于npm （Node Package Manager）是一个包管理器，yum 基于 RPM （RPM Package Manager）。

yum 常用命令：

# 安装
yum install foo-x.x.x.rpm

# 删除
yum remove foo-x.x.x.rpm
# 或者
yum erase foo-x.x.x.rpm

# 更新
yum upgrade foo

# 查看信息
yum info foo

# 搜索（以包含foo字段为例）
yum search foo

# 显示包依赖关系
yum deplist foo

yum 常见参数：

# 静默执行
-e

# 忽略错误
-t

# 设置等待时间
-R[分钟]

# 自动应答（yes）
-y 

# 忽略验证问题
--skip-broken
　
# 忽略GPG验证
--nogpgcheck 

# 检查可更新的包
check-update 

# 清除全部
clearn 

# 清除临时包文件（/var/cache/yum 下文件）
clean packages

# 清除rpm头文件
clearn headers 

# 清除旧的rpm头文件
clean oldheaders 

# 列出包的依赖
deplist 

# 可安装和可更新的RPM包
list 

# 已安装的包
list installed

# 已安装且不在资源库的包
list extras 

# 可安装和可更新的RPM包 信息
info 

# 已安装包的信息(-qa 参数相似)
info installed 

# 安装包
install[RPM包] 

# 安装本地的 RPM包
localinstall

# 更新包
update[RPM包] 

# 升级系统
upgrade 

# 搜索包
search[关键词] 

# 搜索特定包文件名
provides[关键词] 

# 重新安装包
reinstall[RPM包] 

# 显示资源库的配置
repolist

# 指定依赖
resolvedep

# 卸载包
remove[RPM包]
