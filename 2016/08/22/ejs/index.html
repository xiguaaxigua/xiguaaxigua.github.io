ejs模板引擎

08/22/2016 模板引擎

什么是模板引擎

模板引擎（Template Engine）是一个将页面模板和要显示的数据结合起来生成 HTML 页面的工具。

什么是 ejs ?

ejs 是模板引擎的一种，也是我们这个教程中使用的模板引擎，因为它使用起来十分简单，而且与 express 集成良好。

ejs 的标签系统？

  • <% code %>：JavaScript 代码。
  • <%= code %>：显示替换过 HTML 特殊字符的内容。不解析code中的html标签。
  • <%- code %>：显示原始 HTML 内容。解析code中的html标签。

布局

// index.ejs
<%- include a %>
xigua
<%- include b %>

// a.ejs
this is a.ejs

// b.ejs
this is b.ejs

最后输出结果为：

this is a.ejs
xigua
this is b.ejs
