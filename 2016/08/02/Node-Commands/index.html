Node常用命令

2016-08-02
Node.js

安装包

# 全局安装
npm install <name> -g

# 作为项目依赖
npm install <name> --save-dev

新建package.json文件

npm init

更新

npm update <name>

删除

npm remove <name>

卸载

npm uninstall <name>

查看所有全局的包

npm ls

查看当前包的路径

npm root

搜索包

npm search <name>

查看全局包的路径

npm root -g

发布包

npm publish

安装package.json文件中的依赖

#因为node的包相对庞大，不方便加入版本管理，因此将配置信息写去package.json并将其加入版本管理，其他开发者对应下载即可。

# 下载所有的包
npm install

# 只下载dependencies节点的包
npm install --production

清空NPM本地缓存

npm clear cache

撤销发布自己发布过的某个版本的代码

npm unpublish <package>@<version>
