SVN命令

2016-08-24
工具

将文件checkout 到本地

svn checkout svn://192.168.1.6/fldnv/Public/WebSite/branches/0708

向版本库中新增文件

svn add file

将改动的文件提交到版本库

svn commit -m '注释'

加锁/解锁

svn lock -m '注释' 路径

更新到某个版本

svn update -r 版本号 文件路径

查看文件或者目录状态

svn status 路径

删除文件

svn delete path -m '注释'
# 例如：
svn delete svn://192.168.1.1/pro/domain/test.php -m "delete test file"

查看日志

svn log path

查看文件详细信息

svn info path

比较差异

# 将修改文件与基础版本比较
svn diff path

# 比较同一文件版本m和版本n
svn diff -r m:n path

将两个版本之间的差异合并到当前文件夹

svn merge -r m:n path

svn 帮助信息

svn help

查看版本库下的文件和目录

svn list path

创建纳入版本控制的目录

svn mkdir dirName

恢复本地修改

svn revert

解决冲突

# 移除目录或者文件的“冲突状态”
svn resolved
