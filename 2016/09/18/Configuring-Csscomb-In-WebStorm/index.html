WebStorm配置csscomb插件

09/18/2016 工具

什么是csscomb？

官网是这么介绍的：

Make your code beautiful

更准确的说是：Make your css code beautiful。就像下图所示一样，csscomb插件可以格式化你的css代码，看起来更加美观，更利于团队合作，同时运行效率更高。另外，csscomb不仅仅支持css还支持css的预处理器（SASS、LESS、Stylus等）。

csscomb支持Sublime Text 2/3、PhpStorm、WebStorm、IDEA、、PyCharm、RubyMine等IDE。

在Sublime中配置csscomb插件

安装

1.先安装Package Control插件，如果已经安装的话，和安装其他插件一样，搜索csscomb即可。
2.Ctrl + ~调出控制台，复制以下内容到控制台中，回车执行：

# Sublime Text2
import urllib2,os,hashlib; h = '2915d1851351e5ee549c20394736b442' + '8bc59f460fa1548d1514676163dafc88'; pf = 'Package Control.sublime-package'; ipp = sublime.installed_packages_path(); os.makedirs( ipp ) if not os.path.exists(ipp) else None; urllib2.install_opener( urllib2.build_opener( urllib2.ProxyHandler()) ); by = urllib2.urlopen( 'http://packagecontrol.io/' + pf.replace(' ', '%20')).read(); dh = hashlib.sha256(by).hexdigest(); open( os.path.join( ipp, pf), 'wb' ).write(by) if dh == h else None; print('Error validating download (got %s instead of %s), please try manual install' % (dh, h) if dh != h else 'Please restart Sublime Text to finish installation')

# Sublime Text 3
import urllib.request,os,hashlib; h = '2915d1851351e5ee549c20394736b442' + '8bc59f460fa1548d1514676163dafc88'; pf = 'Package Control.sublime-package'; ipp = sublime.installed_packages_path(); urllib.request.install_opener( urllib.request.build_opener( urllib.request.ProxyHandler()) ); by = urllib.request.urlopen( 'http://packagecontrol.io/' + pf.replace(' ', '%20')).read(); dh = hashlib.sha256(by).hexdigest(); print('Error validating download (got %s instead of %s), please try manual install' % (dh, h)) if dh != h else open(os.path.join( ipp, pf), 'wb' ).write(by)

  1. 按 Command + Shift + P ，键入 install 选择 Package Control: Install Package，回车。如下图：

  2. 键入 csscomb ，回车安装即可。

使用方法：
在你的css文件中 Ctrl + A 全选，右键选择 Run Csscomb 即可，当然你也可以自己指定快捷键。

配置

我们可以通过编辑 Settings-User，更改默认的配置。如下图：

Settings-Default中保存了默认的配置。大致配置项如下：

{
    // node的路径
    "node-path" : ":/usr/local/bin",

    // 配置项
    "config": {

        // 属性值后是否添加分号
        "always-semicolon": true,

        // 缩进
        "block-indent": "    ",

        // 颜色值的大小写，例如：#f1f1f1
        "color-case": "lower",

        // 颜色值是否使用缩写，例如：#fff
        "color-shorthand": true,

        // 选择器小写，例如：.demo{}
        "element-case": "lower",

        // csscomb会将属性分组，不同组的属性顺序不同，这个配置是说是否要在不同组的属性之间添加换行以区别他们
        "eof-newline": true,

        // 为0的属性值是否需要单位，例如：top: 0;。
        "leading-zero": false,

        // 单引号还是双引号
        "quotes": "single",

        // Remove all rulesets that contain nothing but spaces.
        "remove-empty-rulesets": true,

        // Set space after `:` in declarations.
        "space-after-colon": " ",

        // Set space after combinator (for example, in selectors like `p > a`).
        "space-after-combinator": " ",

        // Set space after `{`.
        "space-after-opening-brace": "\n",

        // Set space after selector delimiter.
        "space-after-selector-delimiter": " ",

        // Set space before `}`.
        "space-before-closing-brace": "\n",

        // Set space before `:` in declarations.
        "space-before-colon": "",

        // Set space before combinator (for example, in selectors like `p > a`).
        "space-before-combinator": " ",

        // Set space before `{`.
        "space-before-opening-brace": "",

        // Set space before selector delimiter.
        "space-before-selector-delimiter": "",

        // Set space between declarations (i.e. `color: tomato`).
        "space-between-declarations": "\n",

        // Whether to trim trailing spaces.
        "strip-spaces": true,

        // Whether to remove units in zero-valued dimensions.
        "unitless-zero": true,

        // Whether to align prefixes in properties and values.
        "vendor-prefix-align": true,

        // 分组
        "sort-order": [...]
    }
}

在WebStorm中安装csscomb插件

安装

  1. 打开终端，下载csscomb。

     # 下载
 sudo npm install -g csscomb

 # 查看csscomb的版本号，看到版本号即说明安装成功
 csscomb -V

 # 查看csscomb的安装位置，得到的结果填入Program中
 which csscomb // => /usr/local/bin/csscomb

  2. Command + , 打开配置，搜索 External Tools，点击加号添加一个插件。如下图：

  3. 在弹出的编辑框中按照如下填写，在配置Parameters 和 Work directory 选项时，点击右侧 Insert Macro..选择自己需要的配置即可。

  4. 建议不要勾选Open console。

  5. 为csscomb指定一个快捷键，如下图：

配置

在WebStorm中配置csscomb和sublime略有不同，我们需要在项目的目录中新建一个 .csscomb.json 文件，然后写入需要的配置，WebStorm会自动读取这个配置。

比较简单的办法是到csscomb的官网按照提示，可视化的生成一个配置，点击访问，共有24个选项。将生成的配置拷贝到 .csscomb.json 文件中即可。

注意：这里生成的配置中是不包含属性分组这个配置的，我们需要自行添加 sort-order 选项，sort-order 选项是一个二维数组。
