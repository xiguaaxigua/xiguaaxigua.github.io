安装步骤

先安装Homebrew：

/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

利用Homebrew安装mongoDB：

# 更新homebrew
brew update

# 安装mongodb
brew install mongodb

# 启动mongodb
mongodb

利用可视化管理工具Robomongo更好的管理mongodb，这里分享一份mac的破解版本，点击下载，提取密码：u3vq，约10M。
